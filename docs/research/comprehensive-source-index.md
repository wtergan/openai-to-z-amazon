# Comprehensive Source Index

This index expands the shorter source library with the broader OpenAI Z Challenge research corpus staged in the Vault. It is intended for future readers who want to validate the repo narrative, inspect original sources, or continue the research trail.

Notes:

- Sources are grouped by practical role in the project, not by formal citation style.
- `Tier` comes from the Vault-side triage inventory when available.
- `Ingest` reflects the bookmark-ingest state: `summarized`, `metadata-only`, or `unresolved`.
- Sci-Hub links and notes without public URLs are intentionally excluded from the public repo index.
